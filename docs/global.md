# 配置结构

```yaml
log:

api:

ntp:

upstreams:

plugin-matchers:

plugin-executors:

workflows:

listeners:
```

### ```log```

日志配置

### ```api```

```HTTP API``` 配置

### ```ntp```

```NTP``` 服务器配置

### ```upstreams```

上游 ```DNS``` 服务器配置

### ```plugin-matchers```

匹配器插件

### ```plugin-executors```

执行器插件

### ```workflows```

处理流程

### ```listeners```

监听器
