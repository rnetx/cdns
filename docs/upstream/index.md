# Upstream 上游服务器

上游服务器负责向上游服务器转发请求，将上游服务器的响应返回

目前支持以下上游服务器：

- [UDP](udp)
- [TCP](tcp)
- [TLS (DoT | DNS Over TLS)](tls)
- [HTTPS(3) (DoH | DoH3 | DNS Over HTTPS | DNS Over HTTP/3)](https)
- [QUIC (DoQ | DNS Over QUIC)](quic)

- [Parallel](parallel)
- [Random](random)
- [QueryTest](querytest)
- [Hosts](hosts)
